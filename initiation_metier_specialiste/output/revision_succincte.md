# Document Succinct : Révision Générale (Initiation au Métier)

## I. L'Essentiel du Métier
*   **Spécialiste de données** = Pont entre **Affaires** et **Technique**.
*   **Valeur** : Transformer la donnée brute en information stratégique.
*   **Équipe** : Collaboration avec BA, Architectes, Devs et QA.

## II. Méthodologies
*   **Waterfall** (Cascade) : Linéaire, prévisible, mais rigide (effet tunnel).
*   **Agile** : Adaptatif, itératif, centré sur la valeur client.
*   **Cycle de vie** : Analyse -> Conception -> Réalisation -> Tests -> Déploiement.

## III. Outils de Modernité (IA)
*   **Spécialiste Augmenté** : Utilise l'IA pour le nettoyage, le SQL et la documentation.
*   **Prompting** : Rôle + Contexte + Tâche + Contraintes + Format.

## IV. Productivité et Bien-être
*   **Gestion du temps** : Pareto (80/20), Carlson (Continuité), Parkinson (Délais).
*   **Gestion du stress** : Identifier l'agent déclencheur, agir sur les facteurs contrôlables (planification), surveiller les symptômes (cognitifs/physiques).

## V. Éthique et Qualité
*   **Qualité** : Complétude, Validité, Unicité, Cohérence.
*   **Éthique** : Confidentialité, intégrité, respect des cadres légaux.
